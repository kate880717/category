# 🔍 창의 분류 박물관

초등학교 1학년 수학 "분류하기" 단원 학습 도구

## 🌟 라이브 데모
👉 깃허브 페이지 URL을 여기 붙여주세요

## 📦 구조

```
창의 분류 박물관
├── 🌐 GitHub Pages (프론트엔드 = index.html)
│      ↓ fetch
└── ⚙️ Apps Script 웹앱 (백엔드 = Code.gs)
        ↓
   📊 Google Sheets (학생 명단, 문제, 학습 결과)
   📁 Google Drive (녹음 파일)
```

## ✨ 주요 기능

- 📚 **지도안 4활동 완벽 반영** (학용품 → 동물 → 기준 찾기 → 자유 분류)
- 🎤 **사전·사후 음성 녹음** + 한국어 음성 인식 (STT)
- 📁 **Google Drive 녹음 파일 자동 저장** + URL 시트 기록
- 📊 **모든 콘텐츠 시트에서 직접 수정** (학생, 문제, 블록 등)
- 🎨 **저학년 친화 디자인** (큰 글씨, 큰 아이콘, 알록달록)
- 🔊 **스피커 기본 OFF** (켜면 TTS·효과음 ON)
- 💪 **자기평가 별 3개 이하 시 재도전 옵션**
- 🏆 **개인별 임명장 PNG 다운로드**

## 🚀 설치 방법

### 1️⃣ 백엔드 (Apps Script)

1. 새 Google Sheets 만들기
2. 메뉴 **확장 프로그램 → Apps Script**
3. 기본 `Code.gs` 내용 다 지우고 → 본 저장소 `Code.gs` 전체 붙여넣기
4. 저장 (Ctrl+S)
5. 함수 드롭다운에서 `setupSheets` 선택 → **▶️ 실행** → 권한 승인
6. **배포 → 새 배포 → 톱니바퀴 → 웹 앱**
   - 다음 사용자 인증으로 실행: **나**
   - 액세스 권한: **모든 사용자**
7. 배포 → URL 복사

### 2️⃣ 프론트엔드 (GitHub Pages)

1. 본 저장소 또는 새 저장소에 `index.html` 업로드
2. `index.html` 안에서 `GAS_URL` 값을 본인 웹앱 URL로 변경
   ```javascript
   const GAS_URL = "여기에 본인 Apps Script 웹앱 URL";
   ```
3. 저장소 **Settings → Pages → Source: main branch** 설정
4. 몇 분 후 `https://사용자명.github.io/저장소명/` 접속

## 📝 콘텐츠 수정

모두 시트에서 수정 가능 (배포 재실행 X, 즉시 반영)

### 학생명단
| 학번 | 이름 | 반 |
|------|------|-----|
| 1 | 김민준 | 1-1 |

### 활동1·2 (분류 문제)
| ID | 아이콘 | 이름 | 분류기준ID | 분류기준라벨 |
|----|------|------|-----------|-------------|
| i1 | ✏️ | 연필 | pencil | 연필 종류 |

같은 **분류기준ID**를 가진 항목들이 같은 칸으로 분류됨

### 활동3 (기준 찾기)
| ID | 아이콘 | 보기 | 정답여부 | 피드백 |
|----|------|------|---------|---------|
| opt1 | 🎨 | 색깔별로 나누기 | TRUE | 맞아요! ... |

### 활동4 (블록)
- 색깔: `red`, `blue`, `yellow`, `green`
- 모양: `circle`, `square`, `triangle`
- 크기: `big`, `small`

### 사전·사후 질문
| ID | 질문 | 예시이미지설명 |
|----|------|--------------|
| pre1 | 학용품을 어떻게 분류할 수 있을까요? | (참고용) |

## 📊 결과 확인

- **학습결과 시트**: 학생별 점수·자기평가·소요시간 자동 누적
- **녹음기록 시트**: 사전/사후 음성→텍스트 + Drive URL 자동 저장
- **Drive 폴더 `분류박물관_녹음파일`**: 모든 음성 파일 저장

## 🎓 수업 흐름

1. 사전 녹음 (도입)
2. 활동 1·2 (주어진 기준 분류)
3. 활동 3 (분류 기준 탐구)
4. 활동 4 (자유 분류)
5. 사후 녹음 (정리)
6. 자기평가 + 임명장

## ⚠️ 주의사항

- **Chrome / Edge 권장** (음성 인식은 Safari 미지원)
- 마이크 권한 필요 (학생 첫 사용 시 허용 클릭)
- HTTPS 필수 (GitHub Pages는 자동으로 HTTPS)
