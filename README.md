# 🔍 창의 분류 박물관

초등 1학년 수학 「분류하기」 단원 학습 앱입니다.

---

## 📋 구조

- **백엔드**: Google Apps Script (`Code.gs`) → Google Sheets
- **프론트엔드**: GitHub Pages (`index.html` 단일 파일)
- **데이터 저장**: Google Sheets (모든 데이터)
- **음성 녹음**: Google Drive (자동 업로드)
- **STT 텍스트 변환**: 브라우저 Web Speech API → 시트에 자동 기록

---

## 🚀 설치

### 1단계: Google Sheets + Apps Script 설정

1. 새 Google Sheets 만들기
2. **확장 프로그램 → Apps Script** 열기
3. 기본 `Code.gs` 내용 모두 지우고, 본 저장소의 `Code.gs` 내용 전체 붙여넣기
4. 저장 (Ctrl+S)
5. 함수 드롭다운에서 `setupSheets` 선택 → 실행 → 권한 승인
6. **배포 → 새 배포 → 톱니바퀴 → 웹 앱**
   - 다음 사용자 인증으로 실행: **나**
   - 액세스 권한: **모든 사용자**
7. 배포 후 나오는 **웹 앱 URL** 복사

### 2단계: 프론트엔드 설정

1. `index.html`을 GitHub Pages 또는 어떤 정적 호스팅에 업로드
2. `index.html` 안의 `GAS_URL` 변수 값을 1단계의 웹 앱 URL로 변경
3. 브라우저에서 접속 → 끝!

## 🎤 녹음 결과 확인

학생이 녹음 후 자동으로 두 곳에 저장됩니다:

|----------|------|------|------|--------|------|----------|------------|

### 2. Google Drive - `분류박물관_녹음파일` 폴더
- 자동 생성되는 폴더
- 파일명: `{학번}_{이름}_{단계}_{질문ID}_{타임스탬프}.webm`
- 시트의 Drive파일URL 클릭하면 바로 재생 가능

## 🎨 주요 기능

### 음성 / 효과음
- **스피커 기본 OFF**: 처음 시작할 때는 꺼져 있음
- **스피커 켜면**:
  - 모든 화면 안내 자동 음성 (한국어 여성, 유치원 선생님 톤)
  - 정답/오답/성공 효과음
  - 클릭 효과음
  - 모든 질문 음성으로 읽어줌
- **한글 못 읽는 학생도 활동 가능** (스피커만 켜면 됨)

### 자기 평가
- 별 5개 클릭 (회색 → 노란색)
- 별 3개 이하면 다시 도전 기회 제공

### 마이크 권한
- 앱 시작 시 1번만 요청
- localStorage에 영구 저장 → 새로고침해도 다시 안 물어봄

### 반응형
- 태블릿 기본
- 모바일/PC도 자동 적응
- 스크롤 없이 한 화면에 다 보임

## 💡 자주 묻는 질문

**Q. 학생을 더 추가하고 싶어요**
→ `학생명단` 시트에 행만 추가하면 즉시 반영됩니다. 재배포 X.

**Q. 문제를 바꾸고 싶어요**
→ 해당 활동 시트 (`활동1_학용품` 등) 수정. 즉시 반영.

**Q. 음성이 안 나와요**
→ 우측 하단 스피커 아이콘 켜기. 그래도 안 나오면 브라우저가 한국어 음성 미지원.
   Chrome/Edge/Safari 사용 권장.

**Q. 녹음이 안 돼요**
→ 마이크 권한 거부됨. 브라우저 주소창 옆 자물쇠 → 마이크 → 허용.
